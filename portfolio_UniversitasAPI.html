/ Universitas API ExpressJS

Welcome to my portfolio! Here, you can explore my project showcasing a REST API for managing campus data, built using Express.js. This API provides endpoints for retrieving, creating, updating, and deleting various information related to a campus, including students, courses, lectures, and users. The implementation utilizes the powerful features of Express.js, such as routing, middleware, and error handling, to ensure efficient and reliable data management. This API also includes a login feature using JWT token. Feel free to browse through the code and documentation to learn more about the functionalities and the technologies employed in this project

How to use:

// Get list of students

GET /mahasiswa


// Get student details by ID

GET /mahasiswa/:id


// Add a new student

POST /mahasiswa


// Update student

PUT /mahasiswa


// Delete student

Delete /mahasiswa


// Get list of courses

GET /kelas


// Get course details by ID

GET /kelas/:id


// Add a new course

POST /kelas


// Update course

PUT /kelas


// Delete course

Delete /kelas


// Get list of lectures

GET /dosen


// Get lecture details by ID

GET /dosen/:id


// Add a new lecture

POST /dosen


// Update lecture

PUT /dosen


// Delete lecture

Delete /dosen


// Get list of users

GET /users


// Get user details by username

GET /users/:username


// Add a new user

POST /users


// Update user

PUT /users


// Delete user

Delete /users


// Login and Generate JWT

POST /users/login
